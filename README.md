# taxi-booking-system
The Taxi Booking System is a Java Spring Boot project, which allows users to book taxis, calculate fares, and handle billing

Swagger :
```bash
http://localhost:8082/swagger-ui/index.html
```


## Getting Started

Follow these steps to set up and run the Taxi Booking and Billing System.


### Installation

Clone the repository:
```bash
https://github.com/devanand-jrx/taxi-booking-system
```

### Running the Application

#### Navigate to the project directory
cd taxi-booking-system

#### Build the project
mvn clean package


Note : The application will be accessible at ```http://localhost:8082```
